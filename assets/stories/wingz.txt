        "<span class="hl">The merger was a complete clash of cultures</span> and a source of friction              
         as executives from both organizations served their different agendas.             
         As a result, two separate companies were running under the Informix               
         name, a technology-driven UNIX company in Silicon Valley and a PC-focused         
         retail publisher in the farm belt. One former Informix vice president said,       
         <span class="hl">'There was a big rift between Lenexa and Menlo Park. You had people in</span>            
         <span class="hl">Lenexa who thought the people in Menlo Park were strange because they were</span>        
         <span class="hl">Californians. And there were people in California who thought the people in</span>       
         <span class="hl">Kansas were country bumpkins.'</span>"

                WINGZ SPREADSHEET NOW AVAILABLE FOR LINUX                                  
                *****************************************                                  
                                                                                           
Wingz 1.4.1d for Linux has been uploaded to the pub/Linux/Incoming                         
directory at sunsite.unc.edu. It is also available from                                    
ftp://www.wingz.com/pub (see our WWW site: http://www.wingz.com).                          
                                                                                           
Wingz is the award-winning spreadsheet available for Apple Macintosh,                      
MS-Windows, and various Unix X-Windows platforms. Launched by Informix                     
Software, Wingz is now available exclusively from Investment                               
Intelligence Systems Corporation (IISC). It combines standard                              
spreadsheet functionality with advanced business charting and                              
reporting features. A powerful but easy to learn event-driven                              
scripting language, HyperScript, enables the construction of powerful                      
applications employing graphical objects such as buttons, sliders,                         
dialogs, number wheels and the like. Script files may be saved for                         
reuse in multiple applications.                                                            
                                                                                           
Wingz has over 140 built-in functions, a worksheet size of 32k by 32k                      
cells, 20 predefined 2-D and 3-D chart types, multi-font text and                          
graphics image importing. Worksheets and script files are portable                         
between all supported platforms, allowing development and deployment                       
on the most suitable hardware.                                                             
                                                                                           
Wingz for Linux is being released as shareware, with the same power                        
and functionality as its commercial stablemates on other platforms. It                     
is available on the Sunsite Linux archives and directly from                               
ftp.wingz.com, in the /pub directory.                                                      
                                                                                           
For information on the commercial versions of Wingz, email                                 
wingz-i...@wingz.com or see http://www.wingz.com.

TROUBLESHOOTING SECTION                                                                    
                                                                                           
DATABASE ACCESS                                                                            
                                                                                           
If you cannot connect to a database in Wingz Professional or                               
DataLink, you could be encountering a connectivity software                                
restriction that only allows one application to connect to a                               
database at a time. Basically, once Wingz Professional or                                  
DataLink accesses a database, you must log off and exit the                                
application before accessing a database with another                                       
application.  If you don't do this prior to the second access,                             
an error message is displayed indicating that the database or                              
backend is busy or not found.                                                              
                                                                                           
RECOMMENDED RESOURCE SETTINGS WHEN USING Wingz                                             
                                                                                           
The following examples show how the appropriate resources would                            
be set when using the Motif window manager (Mwm).  If instead you                          
are using the Visual User Environment window manager (Vuewm) then                          
replace occurrences of Mwm with Vuewm and if you are using the 4DWm                        
(on an SGI machine) replace occurrences of Mwm with 4DWm.                                  
                                                                                           
If you are using Pointer as the keyboard focus policy, define the following                
resource settings:                                                                         
                                                                                           
Mwm*keyboardFocusPolicy:  pointer                                                          
Mwm*positionOnScreen:  False                                                               
Mwm*clientAutoPlace:  False                                                                
Mwm*autoKeyFocus:  False                                                                   
Mwm*startupKeyFocus:  False                                                                
Mwm*focusAutoRaise:  False                                                                 
Mwm*menu*saveUnder:  True                                                                  
Mwm*positionIsFrame:  True                                                                 
                                                                                           
                                                                                           
If you are using Explicit as the keyboard focus policy, define the following               
resource settings:                                                                         
                                                                                           
Mwm*keyboardFocusPolicy:  explicit                                                         
Mwm*positionOnScreen:  False                                                               
Mwm*clientAutoPlace:  False                                                                
Mwm*autoKeyFocus:  True

OPERATING ENVIRONMENT ANOMALIES                                                            
                                                                                           
Wingz clipboard operations are not functional when you are                                 
running xclipboard.  Exit xclipboard before attempting to perform a cut,                   
copy, or paste operation in Wingz.                                                         
                                                                                           
Wingz is designed to run properly on single-screen X window                                
servers.  You may experience problems (e.g., window painting problems) when                
attempting to run Wingz on a simultaneous multiple-screen                                  
X window server.  In some cases, this can manifest itself as phantom                       
or duplicated menus and palettes.  In other cases, this might cause                        
the Wingz menu to appear confused about what the current                                   
window is.  In other cases, windows might be scattered across multiple                     
sessions.                                                                                  
                                                                                           
You may experience difficulty compiling your external C source code with                   
some compilers.  The difficulty may be attributed to using an ANSI compiler                
or calling incorrect or missing libraries.  The external samples                           
provided are targeted for non-ANSI compilers.  If you are using an ANSI                    
compiler, note that ANSI versions of the makefiles are included.  These                    
files display the makefile name and end with an ".ansi" extension.  To run                 
an ANSI version of the makefile, rename the "makefile.ansi" file to "makefile"             
before building the sample externals.  The C source code is written to be                  
rebuilt for either compiler.                                                               
                                                                                           
                                                                                           
HP 700/800 SPECIFIC ANOMALIES                                                              
                                                                                           
You may encounter a significant delay when attempting to utilize sound                     
commands and functions on a machine that does not have sound capabilities.                 
This could be due to an improperly configured device or the lack of sound                  
capabilities on a particular machine.  Regardless of the reason, the delay                 
can be as much as 15 seconds and is the result of various timeouts that                    
must take place before script execution can continue.                                      
                                                                                           
To run <a class="wingz_" id="wingz_18" target="_blank" href="{{LINK_19}}">Wingz</a> in the background, you may need to execute the                                
command "stty -tostop" in your terminal window prior to launching                          
the application.                                                                           
                                                                                           
                                                                                           
SUN SPECIFIC ANOMALIES                                                                     
                                                                                           
You may see the message:                                                                   
                                                                                           
"Can't Load Codeset file 'C' using internal fallback"                                      
                                                                                           
when starting up the application.  This is caused when the "Codeset"                       
file is not found and it is a warning message only.                                        
                                                                                           
To stop this warning message copy the file $WINGZ3/etc/Codeset into                        
the /usr/lib/X11/C directory (which may need creating).
